感谢您选择易维论坛!

易维论坛是 IP.Board 的汉化版, 修复了原版对中文支持的各种 Bug, 每一句语句的翻译都字斟句酌力求准确, 界面美观大方.

运行环境

推荐运行在 PHP 4.3.0 以上, MySQL 4.0 以上环境.

如果您的服务器不支持 PHP 或 MySQL, 您不能安装易维论坛.

如何安装?

请将'上传目录'下所有文件上传到您的服务器上, 然后打开您的浏览器, 输入'您的论坛地址/install/'.例如, 如果您的论坛安装在 bbs.example.com, 您输入的应该是 bbs.example.com/install/

安装程序将会引导您完成全部安装过程.

日常使用

建议您参阅 IPS 官方撰写的使用手册.地址在欢迎贴子中会给出.
