功能特性

  • 中文化的界面
  • 简明清爽的翻译风格
  • UTF-8 编码,支持多种语言

安装指南

全新安装一份易维论坛非常容易.您只需:
  1. 上传所有文件
  2. 运行 http://您的论坛地址/install/
  3. 进入管理面板,对您的论坛进行初步设置
  4. 完成! 开始使用您的论坛!
